# Currency Converter

Автор: Костюк Никита Иванович

GUI‑приложение для конвертации валют с использованием внешнего API и сохранением истории операций.

## Функционал

* Конвертация между любыми валютами
* Получение актуальных курсов через API
* Сохранение истории последних 10 операций
* Проверка корректности ввода
